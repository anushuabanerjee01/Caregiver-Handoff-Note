# Caregiver Support (Demo, No API Keys)

A simple **Streamlit** caregiver-support demo app that uses **rule-based logic (regex + keywords)** to produce **structured, non-diagnostic** suggestions.

## What this is / isn’t
- ✅ Provides practical caregiving ideas and a notes template (copy/paste).
- ✅ Uses only local rules (no OpenAI, no API keys).
- ❌ Not medical advice, diagnosis, or medication guidance.

## Run locally
1. Create and activate a virtual environment (optional but recommended).
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
